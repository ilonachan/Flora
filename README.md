# Flora
## A modding tool for Professor Layton games

### Supported games
* Professor Layton and the Curious Village: EU

### Current utilities
* GDS script extracting and repacking to a custom readable format - Parameter types 1-3 only

### Requirements
* Python 3
* click (`pip install click`)
